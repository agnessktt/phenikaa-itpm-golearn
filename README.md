# 📚 GoLearn 

## 📖 Project Overview
GoLearn is a web application designed to help students manage and access learning content online.  
The system allows users to register, log in, and browse learning materials in a simple and efficient way.

This project is developed as part of the IT Project Management course using Agile/Scrum methodology.

---

## 🚀 Main Features
- User registration and login  
- View list of learning contents  
- View detailed lesson information  
- User-friendly interface  
- Basic authentication and session management  

---

## 🛠️ Tech Stack
- **Frontend:** React  
- **Backend:** Node.js (Express)  
- **Database:** MySQL  

---

## ⚙️ Installation Guide

### 1. Clone repository
```bash
git clone https://github.com/agnessktt/phenikaa-itpm-golearn.git
cd golearn-app
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the application
```bash
npm start
```
---

## ▶️ Usage
1. Open browser at: http://localhost:3000  
2. Register a new account  
3. Log in to the system  
4. Browse and access learning content
